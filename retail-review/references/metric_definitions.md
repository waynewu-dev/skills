# 零售门店核心经营指标计算公式

## 一、基础指标

### 1. 销售额
```
销售额 = Σ(商品单价 × 销售数量) - 退货金额
```
- **数据来源**：销售明细表
- **统计口径**：按日/周/月汇总，不含税

### 2. 销量
```
销量 = Σ(各SKU销售数量)
```
- **数据来源**：销售明细表
- **统计口径**：按最小销售单位（件/杯/次）汇总

### 3. 成交笔数
```
成交笔数 = 有效交易单据总数
```
- **数据来源**：销售明细表/交易流水
- **统计口径**：同一顾客一次性结账计为1笔

---

## 二、同比/环比指标

### 4. 同比增长率
```
同比增长率 = (当期销售额 - 去年同期销售额) / 去年同期销售额 × 100%
```
- **适用场景**：月度/季度复盘

### 5. 环比增长率
```
环比增长率 = (当期销售额 - 上一周期销售额) / 上一周期销售额 × 100%
```
- **适用场景**：日/周/月度复盘

### 6. 业绩目标达成率
```
目标达成率 = 实际销售额 / 目标销售额 × 100%
```
- **等级判定**：
  - ≥100%：超额完成
  - 90-100%：基本完成
  - 80-90%：差距较小
  - <80%：差距较大

---

## 三、核心三要素指标

### 7. 客流量
```
客流量 = 门店进店总人数（含未成交）
```
- **数据来源**：客流量统计设备或人工计数
- **若无精确数据**：可用成交笔数 / 行业基准转化率反推估算

### 8. 成交转化率
```
转化率 = 成交笔数 / 客流量 × 100%
```
- **行业基准**：参见 retail_knowledge.md 第三部分

### 9. 客单价
```
客单价 = 销售额 / 成交笔数
```
- **波动判断**：同比波动±5%为正常范围

### 10. 连带率（关联购买率）
```
连带率 = 销售总件数 / 成交笔数
```
- 即平均每单购买件数

---

## 四、品类分析指标

### 11. 品类销售占比
```
品类销售占比 = 某品类销售额 / 总销售额 × 100%
```
- 用于识别核心贡献品类和短板品类

### 12. 品类毛利率
```
品类毛利率 = (某品类销售额 - 某品类成本额) / 某品类销售额 × 100%
```

### 13. SKU动销率
```
SKU动销率 = 有销售的SKU数 / 总SKU数 × 100%
```
- ≥70%为健康水平

---

## 五、盈利分析指标

### 14. 毛利率
```
毛利率 = (销售额 - 销售成本) / 销售额 × 100%
```
- **异常判定**：环比下降超3个百分点

### 15. 促销让利占比
```
促销让利占比 = 促销让利总额 / 销售额 × 100%
```
- **安全线**：参见零售行业基准值
- **让利总额** = Σ(原价-促销价) × 促销销量 + 优惠券核销金额 + 满减金额

### 16. 净利率（简易）
```
净利率 ≈ 毛利率 - 固定费用率 - 变动费用率
```
- 如需精确计算，需费用明细数据

---

## 六、会员指标

### 17. 会员消费占比
```
会员消费占比 = 会员消费金额 / 总销售额 × 100%
```

### 18. 会员客单价
```
会员客单价 = 会员消费金额 / 会员成交笔数
```

### 19. 新会员转化率
```
新会员转化率 = 当期新增会员数 / 客流量 × 100%
```

---

## 七、时段分析指标

### 20. 时段销售占比
```
时段销售占比 = 某时段销售额 / 当日总销售额 × 100%
```
- 用于识别高峰/低谷时段，优化排班和营销节奏

---

## 八、复合诊断公式

### 销售额拆解公式（三要素模型）
```
销售额 = 客流量 × 转化率 × 客单价
```
- 任一要素波动，均可通过该公式量化影响程度

### 销售额变动归因
```
Δ销售额 ≈ 客流变动影响 + 转化率变动影响 + 客单价变动影响
```
计算方式：
```
客流影响 = (当期客流 - 基期客流) × 基期转化率 × 基期客单价
转化率影响 = 当期客流 × (当期转化率 - 基期转化率) × 基期客单价
客单价影响 = 当期客流 × 当期转化率 × (当期客单价 - 基期客单价)
```
